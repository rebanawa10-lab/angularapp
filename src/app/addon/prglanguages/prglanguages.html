Program Tools

Angular

• is a popular front-end web framework created by Google. It's used to build dynamic, scalable, single-page web applications (SPAs) offering a complete set of tools (CLI, router, forms, HTTP).

 •A component-based architecture for large, maintainable projects.

 •It emphasizes productivity, code reusability, and best practices, making complex app development more structured and efficient.
Typescript

• TypeScript is a programming language that is a typed superset of JavaScript, primarily used to add static typing to the language.

 • Its main use is to help developers catch errors during development (compile-time) rather than at runtime, which makes codebases more robust, maintainable, and scalable, especially for large projects and teams.
Javascript

• is a versatile, high-level programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS.

 • It is primarily used to add interactivity and dynamic behavior to websites, but its use has expanded to a wide range of applications, including server-side and mobile development.
Docker

• is an open-source software platform that enables developers to easily build, deploy, and run applications in portable, lightweight, self-sufficient environments called containers.

 • Containers package an application's code, runtime, system tools, libraries, and dependencies, ensuring it runs consistently across any environment, from a developer's laptop to production cloud servers.
NodeJS

• an open-source, cross-platform JavaScript runtime environment that allows developers to run JavaScript code outside of a web browser. This enables JavaScript to be used for server-side development, command-line tools, and desktop applications, bridging the gap between front-end and back-end development with a single language.
SQL Server

• Microsoft SQL Server is a proprietary relational database management system (RDBMS) developed by Microsoft. It functions as a database server, storing and retrieving data as requested by other software applications, and is used for a wide range of corporate IT functions, including transaction processing, business intelligence (BI), and analytics.
IIS

• IIS (Internet Information Services) is Microsoft's flexible, secure web server software for Windows, used to host websites, web applications (like ASP.NET), and APIs, handling requests for HTML pages, files, and services over the internet or intranets using protocols like HTTP/HTTPS and FTP.

• It provides robust features for scalability, performance, and isolation through Application Pools, allowing businesses to manage demanding web tasks efficiently.
JSON

• JSON (JavaScript Object Notation) is a lightweight, human-readable, text-based format for storing and exchanging data, using key-value pairs and arrays, making it easy for both humans and machines to understand, widely used in web apps to send data between servers and clients.

• It's language-independent but derived from JavaScript, acting as a universal standard for data interchange in APIs, configuration files, and databases.
Visual Studio Code

• Visual Studio Code (VS Code) is a free app where programmers write and edit code.

 • It's known for its speed, low resource use, and extensive features that streamline coding, making it popular with both beginners and professionals.
C#

• C# (pronounced "C-sharp") is a modern, general-purpose, object-oriented programming language developed by Microsoft as part of the .NET framework.

• It is widely used for building a diverse range of applications and is known for its versatility, robust design, and strong community support.

ANGULAR WORKFLOW

When a user opens your site (e.g. https://example.com):

• MAIN.TS (main switch), the Application Entry Point: 	

        It bootstraps (starts) the Angular app by calling

            script: 
                import  bootstrapApplication  from '@angular/platform-browser';
                import  appConfig  from './app/app.config';
                import  App  from './app/app';
                
                bootstrapApplication(App, appConfig)
                
        It tells Angular to load App, 
        which then loads all components, services, and routing.


• INDEX.HTML (load 1st the browser) The container that wraps your whole Angular app.

        Contains a placeholder tag: < app-root >
        The Root Component, Angular load your app here.

        It also load CSS, icons, fonts, etc.

        The main building block that holds your app's UI and routes.

        Inside it, < app-root > is found.


• APP.TS

        @Component(
            selector: 'app-root'	// The main entrance of the Angular app	
        )
